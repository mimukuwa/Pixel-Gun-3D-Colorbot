## How to use

1. Make sure you have Python installed.

2. Install the required packages:

```bash
pip install numpy mss keyboard pywin32
```

3. Download or clone the repository.

4. Run the script:

```bash
python main.py
```

5. Start Pixel Gun 3D and keep the game as the active window.

## Controls

- `X` - Enable or disable the aim assist
- `F2` - Show or hide the FOV circle
- `Right Ctrl` - Suspend the script

## Configuration

You can change some settings directly in `main.py`:

- `AIMBOT_KEY` - Change the toggle key
- `SENSITIVITY` - Change the color detection sensitivity
- `SMOOTHNESS` - Change the mouse movement smoothness
- `FOV_W` / `FOV_H` - Change the detection area size
